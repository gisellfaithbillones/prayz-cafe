# Fonts Directory

Place your font files here:

## Required Fonts:

### Hatton
- `Hatton-Regular.woff2` (or `.woff`, `.ttf`)
- `Hatton-Medium.woff2` (optional, for medium weight)
- `Hatton-Bold.woff2` (optional, for bold weight)

### FF Providence Sans
- `FFProvidenceSans-Regular.woff2` (or `.woff`, `.ttf`)
- `FFProvidenceSans-Bold.woff2` (optional, for bold weight)

## Where to get the fonts:

- **Hatton**: Available from [Pangram Pangram Foundry](https://pangrampangram.com/products/hatton)
- **FF Providence Sans**: Available from [MyFonts](https://www.myfonts.com/collections/ff-providence-sans-font-fontfont)

Once you add the font files to this directory, they will automatically be loaded by the application.
