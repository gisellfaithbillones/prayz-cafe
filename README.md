# Prayz Cafe

An industrial-style cafe website built with Vue 3 (Options API) and Bootstrap.

## Features

- Vue 3 with Options API
- Bootstrap 5 for responsive design
- Industrial cafe aesthetic
- Menu display with categories

## Setup

Install dependencies:
```bash
yarn install
```

## Development

Run the development server:
```bash
yarn dev
```

## Build

Build for production:
```bash
yarn build
```

## Preview

Preview production build:
```bash
yarn preview
```

## Color Scheme

- Primary Green: `#4b6231`
- Cream: `#f4f2ec`
- Dark Red: `#7a0f1f`
